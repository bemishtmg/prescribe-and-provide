# Welcome to your Lovable project

A High-Trust Digital Pharmacy for Patients & Pharmacists🚀 OverviewThis project is a full-stack web application designed to bridge the gap between patients and pharmacies. It features a secure Prescription Validation workflow and a Virtual Wallet system, ensuring a seamless and reliable medicine procurement process.✨ Key FeaturesDual-Role Authentication: Separate, secure dashboards for Purchasers and Pharmacists.Prescription Portal: Patients can upload prescriptions; Pharmacists can review and validate them in real-time.Virtual Wallet: A local credit-based payment system with Atomic Transactions and Insufficient Funds protection.Live Inventory Sync: Real-time stock updates using Supabase's PostgreSQL backend.Transaction Ledger: A permanent, immutable history of all "purchases" and "wallet resets."🛠️ Tech StackFrontend: React.js, Vite, Tailwind CSS, Shadcn UIBackend/Database: Supabase (PostgreSQL)State Management: React Hooks & Context APIDeployment: Vercel🛡️ Security & ReliabilityRLS (Row Level Security): Ensures users can only see their own prescriptions and wallet balances.Input Validation: Sanitize all medicine quantities and price inputs.Mock Payment Gateway: Simulated bank-grade logic for educational demonstration.📸 Screenshots & DemoLive Demo: [PASTE YOUR VERCEL LINK HERE]Purchaser ViewPharmacist View
